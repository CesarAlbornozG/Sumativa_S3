# Sumativa_S3
Requerimientos:
Funcionales: El programa debe tener el ingreso de datos, debe mostrar los datos ingresados en la consola.
No funcionales: Se debe ejecutar en java, debe ejecutarse el programa sin IDE.

Historias de Usuario
- Realizo el ingreso de datos de un vehiculo, luego reviso la información desplegada en consola.
- Criterios: Deben ingresarse todos los datos, el sistema luego debe mostrar los datos correctamente.
